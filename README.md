# Random Password Generator

**Автор:** Иванов Иван

## Описание

Приложение для генерации случайных паролей с настраиваемыми параметрами и сохранением истории.

### Возможности:
* Настройка длины пароля (4–64 символа)
* Выбор типов символов: цифры, буквы, спецсимволы
* Генерация пароля по нажатию кнопки
* Сохранение истории паролей в JSON‑файл
* Отображение истории в таблице
* Валидация ввода (проверка длины и выбора символов)
